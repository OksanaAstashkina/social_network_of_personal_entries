Основные технологии, которые применялись для front-end разработки сайта или попросту создания его визуальной части (это все, что обрабатывает браузер и выводит на экран): HTML и CSS.

HTML (от англ. HyperText Markup Language) – стандартизированный язык гипертекстовой разметки документов в интернете, который помогает создать основу страниц сайта. Обработанный браузером код отображается уже в привычном для нас виде на компьютере или на мобильных устройствах. Проще говоря, HTML говорит браузеру, о содержании на странице, например, это «заголовок», «список», «элемент списка», «гиперссылка».

CSS (от англ. Cascading Style Sheets – каскадные таблицы стилей) – своеобразное дополнение к языку HTML. CSS представляет собой формальный язык описания внешнего вида документа и отвечает за визуальную составляющую, то есть за дизайн. На этом языке задаются все важные параметры – цветовая гамма элементов, размер шрифтов, отступы, расстояния и т.д. Возможности CSS для визуального оформления сайта переоценить сложно. Благодаря этому языку адаптированные версии веб-ресурсов корректно отображаются на компьютерах и мобильных устройствах. CSS говорит браузеру, как отображать элементы, например, «первый заголовок имеет размер шрифта 30 пикселей и написан шрифтом Verdana» или «все гиперссылки на сайте должны быть красными, без нижнего подчеркивания».

Для back-end разработки сайта, т.е. его программной части использовались: язык программирования Python и фреймворк Django.

Python — это высокоуровневый язык программирования, отличающийся эффективностью, простотой и универсальностью использования. Он широко применяется в разработке веб-приложений и прикладного программного обеспечения, а также в машинном обучении и обработке больших данных. За счет простого и интуитивно понятного синтаксиса является одним из распространенных языков для обучения программированию. Не удивительно, что многие новички с области программирования выбирают именно Python. TIOBE Software представила рейтинг самых популярных языков программирования на август 2022 года. По сравнению с прошлым годом Python прибавил в популярности 3,56%, переместившись со второго на первое место с показателем 15,42%. Многие наиболее успешные технологические компании выбирают Python для создания бэкенда своих продуктов, среди которых: Instagram, Google, Spotify, Netflix, Uber, Dropbox, Pinterest и др. Кстати, интересный факт: язык Python назван его создателем голандцем Гвидо Ван Россум в честь комик-группы Монти Пайтон, а не в честь змеи.

Django — это фреймворк для веб-разработки, написанный на Python в 2005 году. Он позволяет создавать сайты и веб-приложения разной сложности — от личного блога до аудиоплатформы (на нём собран Spotify). От других фреймворков Django отличается модульностью и простотой освоения. Одно из главных преимуществ Django — принцип DRY (don’t repeat yourself): единожды написанный код можно повторно использовать в различных проектах. Поэтому Django часто сравнивают с конструктором Lego. Согласно исследованию Stack Overflow за 2022 год, Django занимает девятое место среди всех веб-фреймворков, обгоняя, например, популярные Spring и Ruby on Rails.На Django работают многие известные сервисы — например, «Инстаграм»*, Pinterest, Spotify, Last.fm и сайт Mozilla. Однако его используют не только в энтерпрайзе: нередко на нём собирают личные сайты: блоги, фотогалереи, портфолио и так далее.

Конечно при создании данного сайта использовались также другие технологии и инструменты, такие как SQL, Bootstrap и многие другие, но мы решили вам рассказать в основных из них.
